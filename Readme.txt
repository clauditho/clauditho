Para iniciar el proyecto se deben seguir los siguientes pasos:

	1. Abrir CMD.
	2. Usar comando cd a la ruta del proyecto (esta se copia tal cual de la barra del navegador de windows) ejemplo: cd C:\User 
	3. Usar el comando "myvenv\Scripts\activate" para iniciar el entorno virtual.
	4. Usar el comando "python manage.py runserver" para iniciar la instancia y esperar que aparezca el siguiente texto:

		Watching for file changes with StatReloader
		Performing system checks...

		System check identified no issues (0 silenced).
		June 03, 2022 - 10:33:58
		Django version 3.2.7, using settings 'proyectoCosechandoDatos.settings'
		Starting development server at http://127.0.0.1:8000/
		Quit the server with CTRL-BREAK.

	5. Ingresar al navegador de internet y colocar la url localhost:8000
